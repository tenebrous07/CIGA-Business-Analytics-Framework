Project Overview

Course: Big Data Computing for Business Analytics (MGT1062)
Team: Thejesh — 23MIA1033
System: Customer Intent Gap Analyzer (CIGA)
Dataset: Multi-Category E-Commerce Behaviour Data (Kaggle)
Source: mkechinov/ecommerce-behavior-data-from-multi-category-store

Architecture

📦 data_loader.py
Kaggle download via kagglehub · Chunked CSV (100K rows/chunk) · Pickle cache · Structured output
🧠 intent_analyzer.py
Intent scoring · Session stitching · Failure classification · Revenue leakage quantification
⚡ app.py
Flask REST API · 8 endpoints · JSON caching · Bootstrap initialisation on startup
🌐 index.html
8-page SPA · Chart.js · Demo mode fallback · Responsive · Real-time alerts

Intent Scoring Formula

intent_score =  (
  total_views       × 1   +
  max_repeat_views  × 3   +
  cart_adds         × 5   +
  comparison_count  × 3   +
  sessions          × 2
) / max_score × 100         # normalised 0 → 100
